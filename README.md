### Primeiros passos
- git init
- git remote add origin => linkar com um repositorio remoto => github
- git pull => pegar as atualizacoes do repo remoto 
- git push => mandar atualizacoes pro remoto

- git checkout -b <nome da branch> => criar a minha branch
